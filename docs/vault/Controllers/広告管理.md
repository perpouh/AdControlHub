# 広告管理コントローラー仕様

## 概要
広告管理システムのコントローラー仕様書です。Web管理画面とAPIの両方を提供します。

## 関連ドキュメント
- [[Models/広告|広告モデル]] - データモデルの詳細仕様
- [[Index|ドキュメント一覧]] - 全体ドキュメントインデックス

## 関連パッケージ
- **Ransack** - 検索・フィルタリング機能
- **Kaminari** - ページネーション機能
- **Rails** - Webフレームワーク
- **PostgreSQL** - データベース（配列型対応）

## ディレクトリ構造
```
app/controllers/
├── advertisements_controller.rb      # Web管理画面用コントローラー
└── api/
    └── v1/
        ├── base_controller.rb        # API基底コントローラー
        └── advertisement_controller.rb # API用コントローラー
```

## Web管理画面コントローラー (AdvertisementsController)

### 基本情報
- **クラス名**: `AdvertisementsController`
- **継承**: `ApplicationController`
- **パス**: `/advertisements`

### アクション一覧

#### 1. index - 広告一覧表示
- **HTTPメソッド**: GET
- **パス**: `/advertisements`
- **機能**: 
  - Ransackを使用した検索・フィルタリング機能
  - ページネーション対応（デフォルト20件、最大100件）
  - 作成日時の降順でソート
- **パラメータ**:
  - `q`: Ransack検索パラメータ
  - `page`: ページ番号
  - `per_page`: 1ページあたりの表示件数（1-100）

#### 2. show - 広告詳細表示
- **HTTPメソッド**: GET
- **パス**: `/advertisements/:id`
- **機能**: 指定された広告の詳細情報を表示

#### 3. new - 広告新規作成フォーム
- **HTTPメソッド**: GET
- **パス**: `/advertisements/new`
- **機能**: 新規広告作成フォームを表示

#### 4. create - 広告作成
- **HTTPメソッド**: POST
- **パス**: `/advertisements`
- **機能**: 新規広告を作成
- **成功時**: 作成された広告の詳細ページにリダイレクト
- **失敗時**: 新規作成フォームを再表示（422エラー）

#### 5. edit - 広告編集フォーム
- **HTTPメソッド**: GET
- **パス**: `/advertisements/:id/edit`
- **機能**: 指定された広告の編集フォームを表示

#### 6. update - 広告更新
- **HTTPメソッド**: PATCH/PUT
- **パス**: `/advertisements/:id`
- **機能**: 指定された広告を更新
- **成功時**: 更新された広告の詳細ページにリダイレクト
- **失敗時**: 編集フォームを再表示（422エラー）

#### 7. destroy - 広告削除
- **HTTPメソッド**: DELETE
- **パス**: `/advertisements/:id`
- **機能**: 指定された広告を削除
- **成功時**: 広告一覧ページにリダイレクト

### プライベートメソッド

#### set_advertisement
- **機能**: 指定されたIDの広告を取得
- **使用箇所**: show, edit, update, destroyアクション

#### advertisement_params
- **機能**: Strong Parametersによるパラメータ制限
- **許可パラメータ**:
  - `title`: 広告タイトル（必須）
  - `link_url`: リンクURL（必須）
  - `image_url`: 画像URL
  - `ad_type`: 広告種別（image/video）
  - `ad_size`: 広告サイズ（vertical/horizontal/square）
  - `alt_text`: 代替テキスト
  - `active`: アクティブ状態
  - `tag_list`: タグリスト（カンマ区切り）

- **クラス名**: `Api::V1::AdvertisementController`

- **パス**: `/api/v1/advertisement`

- **パス**: `/api/v1/advertisement`
- **機能**: 全広告の一覧をJSON形式で返却

```ruby
    resources :advertisement, only: [:index]
  end
end
```

## データモデル連携

### [[Models/広告|Advertisementモデル]]
- **テーブル**: `advertisements`
- **主要フィールド**:
  - `title`: 広告タイトル（255文字制限）
  - `link_url`: リンクURL（URL形式バリデーション）
  - `image_url`: 画像URL（オプション、URL形式バリデーション）
  - `ad_type`: 広告種別（image/video）
  - `ad_size`: 広告サイズ（vertical/horizontal/square）
  - `alt_text`: 代替テキスト
  - `active`: アクティブ状態（デフォルト: false）
  - `tags`: タグ配列（PostgreSQL配列型）

### バリデーション
- `title`: 必須、最大255文字
- `link_url`: 必須、URL形式
- `image_url`: オプション、URL形式
- `ad_type`: 必須、enum値
- `ad_size`: 必須、enum値

詳細なバリデーション仕様は[[Models/広告#バリデーション|広告モデルのバリデーション]]を参照してください。

### スコープ
- `active`: アクティブな広告のみ
- `by_type(type)`: 指定種別の広告
- `by_size(size)`: 指定サイズの広告

詳細なスコープ仕様は[[Models/広告#スコープ|広告モデルのスコープ]]を参照してください。

## セキュリティ考慮事項

### Strong Parameters
- 許可されたパラメータのみを受け付け
- 不正なパラメータの注入を防止

### バリデーション
- サーバーサイドでの厳密なバリデーション
- URL形式の検証
- 文字数制限の適用

## パフォーマンス考慮事項

### ページネーション
- [[#関連パッケージ|Kaminari]]によるページネーション（デフォルト20件、最大100件の制限）
- 大量データの適切な分割表示

### 検索機能
- [[#関連パッケージ|Ransack]]による効率的な検索・フィルタリング
- インデックスを活用したクエリ最適化

## エラーハンドリング

### HTTPステータスコード
- 200: 成功
- 422: バリデーションエラー
- 404: リソース未発見

### ユーザーフィードバック
- 成功時: 日本語の成功メッセージ
- 失敗時: バリデーションエラーの表示
