# 広告モデル (Advertisement)

## 概要
広告を管理するためのモデルです。画像・動画の広告をサポートし、タグ機能とアクティブ状態の管理を行います。

## データベーススキーマ

### テーブル名: `advertisements`

| カラム名 | データ型 | NULL | デフォルト値 | 説明 |
|---------|---------|------|-------------|------|
| id | bigint | NOT NULL | - | 主キー |
| title | string(255) | NOT NULL | - | 広告タイトル |
| link_url | string(255) | NOT NULL | - | 広告のリンク先URL |
| image_url | string(255) | NULL | - | 広告画像のURL |
| active | boolean | NULL | false | アクティブ状態 |
| tags | string[] | NULL | [] | タグ配列 |
| ad_type | string | NOT NULL | 'image' | 広告種別 |
| ad_size | string | NOT NULL | 'square' | 広告サイズ |
| alt_text | string | NULL | - | 代替テキスト |
| created_at | datetime | NOT NULL | - | 作成日時 |
| updated_at | datetime | NOT NULL | - | 更新日時 |

## フィールド詳細

### 必須フィールド

#### 1. 広告種別 (ad_type)
- **必須**: はい
- **デフォルト値**: 'image'
- **選択肢**:
  - `image`: 画像広告
  - `video`: 動画広告
- **バリデーション**: enum値のみ許可

#### 2. 広告サイズ (ad_size)
- **必須**: はい
- **デフォルト値**: 'square'
- **選択肢**:
  - `vertical`: 縦長
  - `horizontal`: 横長
  - `square`: 正方形
- **バリデーション**: enum値のみ許可

#### 3. URL (link_url)
- **必須**: はい
- **最大長**: 255文字
- **バリデーション**: 有効なURL形式である必要があります

#### 4. タグ (tags)
- **必須**: はい
- **データ型**: 文字列配列
- **デフォルト値**: 空配列 `[]`
- **バリデーション**: 文字列の配列である必要があります

### 任意フィールド

#### 1. 代替テキスト (alt_text)
- **必須**: いいえ
- **最大長**: 制限なし
- **用途**: アクセシビリティ対応

#### 2. 画像URL (image_url)
- **必須**: いいえ
- **最大長**: 255文字
- **バリデーション**: 有効なURL形式である必要があります（空の場合はスキップ）

#### 3. アクティブ状態 (active)
- **必須**: いいえ
- **デフォルト値**: false
- **用途**: 広告の表示/非表示制御

## バリデーション

### 基本バリデーション
- `title`: 必須、最大255文字
- `link_url`: 必須、有効なURL形式
- `image_url`: 任意、有効なURL形式（空の場合はスキップ）
- `ad_type`: 必須、enum値のみ
- `ad_size`: 必須、enum値のみ
- `tags`: 必須、文字列配列

### カスタムバリデーション
- `tags_must_be_array_of_strings`: タグが文字列の配列であることを確認

## コールバック

### before_validation
- `normalize_tags`: タグの正規化
  - 文字列の場合は配列に変換
  - 配列の場合は空文字列を除去

## スコープ

### フィルタリング用スコープ
- `active`: アクティブな広告のみを取得
- `by_type(type)`: 指定した種別の広告を取得
- `by_size(size)`: 指定したサイズの広告を取得

## インスタンスメソッド

### タグ関連
- `tag_list`: タグをカンマ区切り文字列で取得
- `tag_list=(value)`: カンマ区切り文字列からタグ配列を設定

### 表示用
- `display_name`: 表示用の名前（例: "広告タイトル (Image - Square)"）

### 種別判定
- `image?`: 画像広告かどうか
- `video?`: 動画広告かどうか

### サイズ判定
- `vertical?`: 縦長かどうか
- `horizontal?`: 横長かどうか
- `square?`: 正方形かどうか

## 使用例

### 広告の作成
```ruby
# 画像広告（正方形）
ad = Advertisement.create!(
  title: "新商品キャンペーン",
  link_url: "https://example.com/campaign",
  image_url: "https://example.com/image.jpg",
  ad_type: "image",
  ad_size: "square",
  alt_text: "新商品の広告画像",
  tags: ["キャンペーン", "新商品", "限定"],
  active: true
)

# 動画広告（横長）
video_ad = Advertisement.create!(
  title: "プロモーション動画",
  link_url: "https://example.com/video",
  image_url: "https://example.com/video-thumb.jpg",
  ad_type: "video",
  ad_size: "horizontal",
  tags: ["動画", "プロモーション"],
  active: false
)
```

### タグの設定
```ruby
# 配列で設定
ad.tags = ["タグ1", "タグ2", "タグ3"]

# カンマ区切り文字列で設定
ad.tag_list = "タグ1, タグ2, タグ3"

# タグの取得
ad.tag_list # => "タグ1, タグ2, タグ3"
```

### スコープの使用
```ruby
# アクティブな広告のみ
Advertisement.active

# 画像広告のみ
Advertisement.by_type("image")

# 縦長の広告のみ
Advertisement.by_size("vertical")

# 組み合わせ
Advertisement.active.by_type("image").by_size("square")
```

## 注意事項

1. **URL形式**: `link_url`と`image_url`は有効なURL形式である必要があります
2. **タグ配列**: タグは文字列の配列として保存され、空文字列は自動的に除去されます
3. **デフォルト値**: `ad_type`と`ad_size`にはデフォルト値が設定されているため、明示的に指定しない場合は自動的に設定されます
4. **アクティブ状態**: 新規作成時は`active: false`がデフォルトです
